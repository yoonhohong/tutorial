Tutorial

Data Clinical Medicine

개요
인공지능의 시대에 의학 연구 및 의료 현장에서 기계학습을 이용한 데이터 분석 역량이 중요시되고 있다. 본 과정에서는 전세계적으로 가장 널리 사용되는 오픈소스 프로그램인 R을 이용해 데이터 가공과 시각화, 통계적 모델링과 기계학습기법을 학습한다. 실제 임상 데이터를 이용하여 프로그래밍을 직접 수행하며, 데이터 의학에 필요한 기본적인 데이터 분석능력을 갖추는 것을 목적으로 한다.

목표
1. 의학/의료 데이터의 종류와 구조를 이해하고, 데이터 의학의 개념과 전망을 설명할 수 있다. 2. R의 기본 문법을 이해하고, 주요 패키지를 활용해 탐색적 데이터 분석과 통계적 모델링을 할 수 있다. 3. 기계학습의 기본 개념과 작업 과정, 주요 알고리즘을 이해하고, 데이터 분석에 활용할 수 있다.

교재
An Introduction to Statistical Learning (with Applications in R), Springer (Gareth James, Daniel Witten, Trevor Hastie, Robert Tibshirani, 2013)
