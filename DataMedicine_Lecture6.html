Cross validation

모델의 성능 평가(model assessment)
적절한 수준의 flexibility 를 선택(model selection)하는 데 사용

Validation set approach
test error 에 대한 추정치와 추정치의 변동성이 크다 (즉, bias와 variance가 모두 크다)
LOOCV (Leave-one-out cross-validation)
test error 에 대한 추정치와 추정치의 변동성이 훨씬 작다 (즉, bias와 variance가 모두 작다)
k-fold CV
test error 에 대한 추정치의 정확성, 즉 bias는 validation set approach와 LOOCV의 중간에 해당
test error 추정치는 LOOCV의 추정치보다 변동성이 작은 경향이 있다.
rule of thumb: k=5 or 10 (bias-variance trade-off)

validation set approach
평균제곱오차의 추정치를 구해보자. 
mse = c()
for (i in 1:10){
  train = sample(392, 196)
  lm.fit = lm(mpg ~ horsepower, data = Auto, subset = train)
  pred = predict(lm.fit, Auto[-train,])
  obs = Auto[-train,]$mpg
  mse[i] = mean((pred - obs)^2)
}

Bootstrap

추정치(parameter estimation)의 정확도(변동성) 평가

표본평균에서 모평균을 추정하는 문제… 먼저, remind…what you learned in high school…

# population 
population = rnorm(1000, 0, 10) # number=1000, mean=0, sd=10
mean(population)
## [1] -0.2908895
sd(population)
## [1] 10.46025
# sample 
spl = sample(population, 100, replace = F)

suppose that we don’t know the population mean and sd,
estimate the population mean
using the sample data

중심극한정리에 따르면… \[\mu = mean(\bar{X})\] \(\mu: population\ mean\)
\(\bar{X}: sample\ mean\)

standard error (of mean) = standard deviation of estimated population mean \[SE=\frac{\sigma}{n}\]

\(\sigma = standard\ deviation\ of\ population\) \(n = sample\ size\)

95% confidence interval of population mean \[\bar{X}-1.96\times\frac{\sigma}{\sqrt{n}} < \mu < \bar{X}+1.96\times\frac{\sigma}{\sqrt{n}}\]

we can replace \(\sigma\) with \(\hat{\sigma}\)
\[\hat{\sigma} = s\] \(s = standard\ deviation\ of\ sample\)

We can calculate standard deviation of sample as following:
\[s^2 = \frac{\sum(X_i-\bar{X})^2}{n-1}\] note: n-1 instead of n

표본평균에서 모평균을 추정하는 문제로 돌아가서…
Calculate sample mean and sd -> Estimate population mean

x_bar = mean(spl)
